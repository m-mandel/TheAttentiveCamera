USE DIRECTIONS:
1. Run program, a window (titled "background") should appear.
2. When background is clear (e.g. no people or moving objects are in the frame), 
	press spacebar to start displaying live feed.
3. Should detect faces and hands, when hand location is detected on top of a detected face -
	the person is (approximately) cropped out.
4. To exit program: press Esc.

Requirements:
1. Python 2.7
2. OpenCV library.
3. webcam connected to computer (may need to change to appropriate number in code).
